# 1. Instalación

```bash
> git clone https://github.com/rogerforner/blank-chrome-ext.git
> cd blank-chrome-ext
> npm install
```

**Dependencias**

> _package.json_

```json
"devDependencies": {
  "laravel-mix": "^5.0.1"
}
```
